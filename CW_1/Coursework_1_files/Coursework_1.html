Introduction

In this assignment you will implement a decision tree algorithm and then use it for bagging and boosting. We've provided a tree structure for you with distinct leaves and nodes. Leaves have two fields, parent (another node) and prediction (a numerical value). Nodes have six fields:

  1. left: node describing left subtree
  2. right: node describing right subtree
  3. parent: the parent of the current subtree. The head of the tree always has None as its parent. Feel free to initialize nodes with this field set to None so long as you set the correct parent later on.
  4. cutoff_id: index of feature to cut
  5. cutoff_val: cutoff value c (<=c : left, and >c : right)
  6. prediction: scalar prediction at this node

def spiraldata(N=300):
    r = np.linspace(1,2*np.pi,N)
    xTr1 = np.array([np.sin(2.*r)*r, np.cos(2*r)*r]).T
    xTr2 = np.array([np.sin(2.*r+np.pi)*r, np.cos(2*r+np.pi)*r]).T
    xTr = np.concatenate([xTr1, xTr2], axis=0)
    yTr = np.concatenate([np.ones(N), -1 * np.ones(N)])
    xTr = xTr + np.random.randn(xTr.shape[0], xTr.shape[1])*0.2
    
    xTe = xTr[::2,:]
    yTe = yTr[::2]
    xTr = xTr[1::2,:]
    yTr = yTr[1::2]
    
    return xTr,yTr,xTe,yTe

First, implement the function sqsplit which takes as input a (weighted) data set with labels and computes the best feature and cut-value of an optimal split based on minimum squared error. The third input is a weight vector which assigns a positive weight to each training sample. The loss you should minimize is the averaged weighted squared-loss: [Math Processing Error]L(S)=iLwi(yiTL)2+iRwi(yiTR)2.

You are building a regression tree, and right now you need to choose a split for the given dataset [Math Processing Error]S={(x1,y1),,(xn,yn)} (where we have continuous labels [Math Processing Error]yiR). Suppose you split on some feature [Math Processing Error]j with value [Math Processing Error]c and partition the dataset in to two sets of indices, [Math Processing Error]L--the set of indices on the left (i.e., [Math Processing Error]iL[xi]j<c)--and [Math Processing Error]R--the set of indices on the right (i.e., [Math Processing Error]iR[xi]j>c). Suppose you assign every data point on the left the prediction [Math Processing Error]TL and every data point on the right the prediction [Math Processing Error]TR. Finally, suppose that each data point [Math Processing Error]xi has an associated weight [Math Processing Error]wi, and that the weights are normalized (i.e., [Math Processing Error]iwi=1).

First, we show that setting [Math Processing Error]TL and [Math Processing Error]TR to the weighted average label over their respective sets (i.e., [Math Processing Error]TL=1WLiLwiyi and [Math Processing Error]TR=1WRiRwiyi) minimizes the loss [Math Processing Error]L, where [Math Processing Error]WL=iLwi and [Math Processing Error]WR=iRwi are the total weight of the left and right side respectively.

We take the derivative of the loss with respect to [Math Processing Error]TL to obtain [Math Processing Error]ddTLL(S)=2iLwi(yiTL)=2iLwiyi+2TLiwi Setting this equal to zero and solving, we get [Math Processing Error]2TLwL=2iLwiyi and therefore [Math Processing Error]TL=1WLiLwiyi A symmetric argument holds for [Math Processing Error]TR.

Now, imagine you are considering splitting on some feature [Math Processing Error]j, and suppose you have already sorted the training points in the order of this feature value, so that [Math Processing Error][x1]j<[x2]j<<[xn]j. You'd like to choose a split from among [Math Processing Error]c1c2cn1, where [Math Processing Error]ci=[xi]j+[xi+1]j2. One way to do this would be to, for each possible split [Math Processing Error]ck, decide whether each [Math Processing Error]xi should be partitioned left or right, and compute [Math Processing Error]L. At the end, take the split with the lowest loss. The number of data points [Math Processing Error]n and there are [Math Processing Error]O(n) splits to consider, and the proposed algorithm would require [Math Processing Error]O(n) per split to evaluate [Math Processing Error]L, for a total of [Math Processing Error]O(n2) time.

Now, suppose some split [Math Processing Error]ck results in the data being partitioned in to [Math Processing Error]L(k) and [Math Processing Error]R(k). Suppose you are given the following quantities precomputed: [Math Processing Error]WL(k), [Math Processing Error]PL(k)=iLwiyi, and [Math Processing Error]QL(k)=iLwiyi2. Similarly, you are given [Math Processing Error]WR(k), [Math Processing Error]PR(k) and [Math Processing Error]QR(k) Equipped with these precomputed quantities, we can compute [Math Processing Error]L in constant time:

Expand the left side of the loss to [Math Processing Error]iLwiyi22iLwiyiTL+iLwiTL2. The first term is exactly [Math Processing Error]QL(k). The second term can be written as [Math Processing Error]2PL(k)PL(k)WL(k)=2PL(k)2wL(k). The last term can be written as [Math Processing Error]wL(k)PL(k)2wL(k)2=PL(k)2wL(k). The second term plus the third term is therefore simply [Math Processing Error]PL(k)2wL(k). Therefore the whole expression can be evaluated as: [Math Processing Error]QL(k)PL(k)2wL(k) Similarly, the right term is: [Math Processing Error]QR(k)PR(k)2wR(k)

Efficent Update Rule: If all feature values are distinct, only one data point moves from [Math Processing Error]R to [Math Processing Error]L when moving from split [Math Processing Error]k to split [Math Processing Error]k+1. Therefore, we simply update the values accordingly. For example, we subtract [Math Processing Error]wk from [Math Processing Error]WR(k) and add it to [Math Processing Error]WL(k). We subtract [Math Processing Error]wkyk from [Math Processing Error]PR(k) and add it to [Math Processing Error]PL(k). We subtract [Math Processing Error]wkyk2 from [Math Processing Error]QR(k) and add it to [Math Processing Error]QL(k). Crucially, all of these updates take only constant time.

def sqsplit(xTr,yTr,weights=[]):
    """Finds the best feature, cut value, and loss value.
    
    Input:
        xTr:     n x d matrix of data points
        yTr:     n-dimensional vector of labels
        weights: n-dimensional weight vector for data points
    
    Output:
        feature:  index of the best cut's feature
        cut:      cut-value of the best cut
        bestloss: loss of the best cut
    """
    N,D = xTr.shape
    assert D > 0 # must have at least one dimension
    assert N > 1 # must have at least two samples
    if weights == []: # if no weights are passed on, assign uniform weights
        weights = np.ones(N)
    weights = weights/sum(weights) # Weights need to sum to one (we just normalize them)
    bestloss = np.inf
    feature = np.inf
    cut = np.inf
    
    for i in range(D):
        # initialize the values accordingly for each loop
        W_l, W_r = 0, 1
        P_l, P_r = 0, weights@yTr
        Q_l, Q_r = 0, weights@(yTr**2)
        index = np.argsort(xTr[:, i])
        yTr_i, weights_i = yTr[index], weights[index]
        for j in range(N-1):
            W_incre = weights_i[j]
            P_incre = W_incre * yTr_i[j]
            Q_incre = W_incre * (yTr_i[j]**2)
            W_l, W_r = W_l + W_incre, W_r - W_incre
            P_l, P_r = P_l + P_incre, P_r - P_incre
            Q_l, Q_r = Q_l + Q_incre, Q_r - Q_incre
            LS = Q_l-((P_l**2)/W_l) + Q_r-((P_r**2)/W_r)
            # only update for distinct feature values
            if LS < bestloss and xTr[index][j,i] != xTr[index][j+1,i]:
                feature = i
                cut = (xTr[index][j,i] + xTr[index][j+1,i])/2
                bestloss = LS
    return feature, cut, bestloss

def cart(xTr,yTr,depth=np.inf,weights=None):
    """Builds a CART tree.
    
    The maximum tree depth is defined by "maxdepth" (maxdepth=2 means one split).
    Each example can be weighted with "weights".

    Args:
        xTr:      n x d matrix of data
        yTr:      n-dimensional vector
        maxdepth: maximum tree depth
        weights:  n-dimensional weight vector for data points

    Returns:
        tree: root of decision tree
    """
    n,d = xTr.shape
    if weights is None:
        w = np.ones(n) / float(n)
    else:
        w = weights
    
    tree_ct = TreeNode(None, None, None, None, None, None)
    # the case for depth is 1 or yTr are the same
    if depth == 1 or np.all(yTr == yTr[-1]):
        tree_ct.prediction = (w@yTr)/np.sum(w)
        return tree_ct
    feature, cut, bestloss = sqsplit(xTr, yTr, w)
    # the case for feature is inf
    if feature == np.inf:
        tree_ct.prediction = (w@yTr)/np.sum(w)
        return tree_ct
    x_tr = xTr[:, feature]
    index_l = np.where(x_tr <= cut)[0]
    index_r = np.delete(np.arange(n), index_l)
    tree_ct.cutoff_id = feature
    tree_ct.cutoff_val = cut
    tree_ct.left = cart(xTr[index_l], yTr[index_l], depth-1, w[index_l])
    tree_ct.right = cart(xTr[index_r], yTr[index_r], depth-1, w[index_r])
    tree_ct.left.parent = tree_ct
    tree_ct.right.parent = tree_ct
    return tree_ct

# process single data point
def process(x_i, root):
    cut = root.cutoff_val
    feature = root.cutoff_id
    pred = root.prediction
    if cut != None:
        if x_i[feature] <= cut:
            root_new = root.left
            return process(x_i, root_new)
        else:
            root_new = root.right
            return process(x_i, root_new)
    else:
        return  pred

def evaltree(root,xTe):
    """Evaluates xTe using decision tree root.
    
    Input:
        root: TreeNode decision tree
        xTe:  n x d matrix of data points
    
    Output:
        pred: n-dimensional vector of predictions
    """
    assert root is not None
    
    n = xTe.shape[0]
    pred = np.zeros(n)
    for i in range(n):
        pred[i] = process(xTe[i], root)
    return pred

def forest(xTr, yTr, m, maxdepth=np.inf):
    """Creates a random forest.
    
    Input:
        xTr:      n x d matrix of data points
        yTr:      n-dimensional vector of labels
        m:        number of trees in the forest
        maxdepth: maximum depth of tree
        
    Output:
        trees: list of TreeNode decision trees of length m
    """
    
    n, d = xTr.shape
    trees = []
    
    for i in range(m):
        index = np.random.randint(n, size = n)
        tree_i = cart(xTr[index], yTr[index], maxdepth)
        trees.append(tree_i)
    return trees

def evalforest(trees, X, alphas=None):
    """Evaluates X using trees.
    
    Input:
        trees:  list of TreeNode decision trees of length m
        X:      n x d matrix of data points
        alphas: m-dimensional weight vector
        
    Output:
        pred: n-dimensional vector of predictions
    """
    m = len(trees)
    n,d = X.shape
    if alphas is None:
        alphas = np.ones(m) / len(trees)
            
    pred = np.zeros(n)
    
    # create a matrix to store all preds
    pred_mat = np.zeros((m, n))
    for i in range(m):
        pred_mat[i] = evaltree(trees[i], X)
        
    pred = pred_mat.T @ (alphas/np.sum(alphas))
    return pred

Boosting

Another option to improve your decision trees is to build trees of small depth (e.g. only depth=3 or depth=4). These do not have high variance, but instead suffer from high bias. You can reduce the bias of a classifier with boosting. Implement the function boosttree, which applies Adaboost on your cart functions. You should be able to use the function evalforest to evaluate your boosted ensemble (provdided you pass on the weights correctly.)

def boosttree(x,y,maxiter=100,maxdepth=2):
    """Learns a boosted decision tree.
    
    Input:
        x:        n x d matrix of data points
        y:        n-dimensional vector of labels
        maxiter:  maximum number of trees
        maxdepth: maximum depth of a tree
        
    Output:
        forest: list of TreeNode decision trees of length m
        alphas: m-dimensional weight vector
        
    (note, m is at most maxiter, but may be smaller,
    as dictated by the Adaboost algorithm)
    """
    assert np.allclose(np.unique(y), np.array([-1,1])); # the labels must be -1 and 1 
    n,d = x.shape
    weights = np.ones(n) / n
    preds   = None
    forest  = []
    alphas  = []

    for i in range(maxiter):
        tree_i = cart(x, y, maxdepth, weights)
        forest.append(tree_i)
        y_hat = evaltree(tree_i, x)
        eps = (abs(y_hat - y) @ weights)/2
        # epsilon < 1/2 all the time
        alpha = np.log((1-eps)/eps)/2
        alphas.append(alpha)
        weights = weights*np.exp(-alpha*(y_hat*y))
        weights = weights/(2*np.sqrt(eps-eps**2))

    return forest, alphas

def evaltreecomp(root,xTe,idx=[]):
    """Evaluates xTe using decision tree root. Same as evaltree but designed to be as efficient as possible.
    
    Input:
        root: TreeNode decision tree
        xTe:  n x d matrix of data points
        idx:  choosen indices, optional argument that might be helpful with implementation strategy
    Output:
        pred: n-dimensional vector of predictions
    """
    assert root is not None
    n = xTe.shape[0]
    pred = np.zeros(n)
    
    for i in range(n):
        pred[i] = process(xTe[i], root)
    return pred

def cartcomp(xTr,yTr,depth=np.inf,weights=None):
    """Builds a CART tree. Same as cart but designed to be as efficient as possible.
    
    The maximum tree depth is defined by "maxdepth" (maxdepth=2 means one split).
    Each example can be weighted with "weights".

    Args:
        xTr:      n x d matrix of data
        yTr:      n-dimensional vector
        maxdepth: maximum tree depth
        weights:  n-dimensional weight vector for data points

    Returns:
        tree: root of decision tree
    """
    n,d = xTr.shape
    if weights is None:
        w = np.ones(n) / float(n)
    else:
        w = weights
    
    tree_ct = TreeNode(None, None, None, None, None, None)
    # the case for depth is 1 or yTr are the same
    if depth == 1 or np.all(yTr == yTr[-1]):
        tree_ct.prediction = (w@yTr)/np.sum(w)
        return tree_ct
    feature, cut, bestloss = sqsplit(xTr, yTr, w)
    # the case for feature is inf
    if feature == np.inf:
        tree_ct.prediction = (w@yTr)/np.sum(w)
        return tree_ct
    x_tr = xTr[:, feature]
    index_l = np.where(x_tr <= cut)[0]
    index_r = np.delete(np.arange(n), index_l)
    tree_ct.cutoff_id = feature
    tree_ct.cutoff_val = cut
    tree_ct.left = cart(xTr[index_l], yTr[index_l], depth-1, w[index_l])
    tree_ct.right = cart(xTr[index_r], yTr[index_r], depth-1, w[index_r])
    tree_ct.left.parent = tree_ct
    tree_ct.right.parent = tree_ct
    return tree_ct
